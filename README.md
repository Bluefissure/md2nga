# md2nga

Python script to convert Markdown to NGA BBS Code

## Usage

`python md2nga.py -i README.md -o README.nga`

`python md2nga.py -h`

## Feature

- [x] Bold, italic, delete
- [x] Hyperlink
- [x] Image link
- [x] Unordered list
- [ ] Ordered list
- [ ] Table

## Hint

1. NGA only have two level heads, indentify your default base head level via `-hl`
